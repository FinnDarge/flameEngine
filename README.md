# Flame Game

A Flutter game built with Flame Engine.

## Getting Started

This project uses [Flame Engine](https://docs.flame-engine.org/latest/) for game development.

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK

### Installation

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run the game:
   ```bash
   flutter run
   ```

## Project Structure

- `lib/main.dart` - Entry point with game initialization
- `lib/game/` - Game logic and components
- `assets/images/` - Image assets
- `assets/audio/` - Audio assets
